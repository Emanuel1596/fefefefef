## Cambios clave

- Se eliminaron los formatos de codigo en linea para evitar recuadros grises en GitHub.
- Se agregaron anclas HTML en el indice de requerimientos para que los enlaces funcionen al dar clic.
- Se actualizo la compra para permitir varios cursos en una misma orden.
- Se dejo claro que Orden guarda la compra completa y Orden_Detalle guarda cada curso comprado.
- Se dejo Inscripcion conectada con Orden_Detalle para que cada curso comprado genere su propio acceso educativo.
- Las explicaciones extra del modelo de base de datos se dejaron como explicaciones, no como reglas recomendadas.
- La estructura de carpetas ahora representa el proyecto completo finalizado, no solo el repositorio actual.
