¿Qué vas a poder ver en este curso?

En este curso revisarás conceptos básicos de seguridad informática, reconocimiento visual de entornos digitales y análisis general de riesgos dentro de una aplicación web. La intención es que el alumno pueda comprender el tema con lenguaje sencillo, ejemplos claros y una ruta organizada por módulos y lecciones.

El curso está pensado para estudiantes que ya conocen elementos básicos de computación y desean subir al siguiente nivel para entender cómo se protege el acceso a la información dentro de una plataforma.

Conocimientos previos necesarios (recomendados)

  • Conceptos básicos de navegación web.
  • Uso general de formularios, usuarios y contraseñas.
  • Conocimientos básicos de HTML, CSS y JavaScript.
